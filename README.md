## Descrição

Este repositório contém os códigos desenvolvidos ao longo da disciplina de **Algoritmos e Estruturas de Dados III**.  
Os projetos foram implementados utilizando a linguagem **Prolog**, com foco em **programação lógica**, construção de **bases de conhecimento** e resolução de problemas através de **inferência lógica**.

## 🧩 O que contém

- Exercícios e projetos da disciplina **AEDS III** implementados em **Prolog**.
- Implementações utilizando **fatos, regras e consultas**.
- Problemas resolvidos com **recursão, listas e backtracking**.
- Exemplos de **bases de conhecimento** e sistemas simples de inferência.

### Recursos do Repositório:

- **Programação Lógica:**  
  Implementações utilizando o paradigma lógico para representar conhecimento e resolver problemas.

- **Consultas e Inferência:**  
  Uso de consultas (*queries*) para extrair informações a partir das regras definidas.

- **Organização Estruturada:**  
  Códigos separados por pastas para facilitar navegação e manutenção.

  - `exercicios`: Contém exercícios desenvolvidos durante a disciplina.
  - `projetos`: Trabalhos práticos desenvolvidos ao longo do curso.
