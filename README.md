# 📚 Prolog Learning

Repositório dedicado ao estudo da linguagem Prolog, explorando conceitos fundamentais de programação lógica, resolução de problemas e construção de bases de conhecimento.

# 📖 Sobre

Este repositório contém exercícios, exemplos e implementações desenvolvidas durante o aprendizado da linguagem Prolog. O objetivo é praticar conceitos como:

Fatos e regras

Consultas (queries)

Recursão

Listas

Backtracking

Resolução lógica
